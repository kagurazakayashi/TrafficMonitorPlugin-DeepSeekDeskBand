# Ассистент DeepSeek (DeepSeekDeskBand)

![screenshot](screenshot.png)

Плагин для десктопного помощника [TrafficMonitor](https://github.com/zhongyang219/TrafficMonitor). Отображает баланс DeepSeek в реальном времени в окне панели задач, с поддержкой истории изменений баланса.

## Возможности

- Периодический запрос баланса через официальное API DeepSeek
- Отображение суммы баланса и валюты в панели задач и главном окне TrafficMonitor
- История изменений баланса (автоматическое инкрементное сохранение, настраиваемый лимит)
- Поддержка нескольких языков (автоопределение системного языка или ручной выбор)
- Файл конфигурации (включая API Key) и история шифруются с помощью Windows DPAPI, доступны только текущему пользователю Windows

## Установка

1. Скачайте соответствующую версию DLL со страницы [Release](https://github.com/KagurazakaYashi/TrafficMonitorPlugin-DeepSeekDeskBand/releases):
   - TrafficMonitor x86 → `DeepSeekDeskBand.dll`
   - TrafficMonitor x64 → `DeepSeekDeskBand64.dll`
2. Скопируйте DLL в папку `plugins` в каталоге установки TrafficMonitor
3. Перезапустите TrafficMonitor
4. Щёлкните правой кнопкой мыши по пустой области панели задач, выберите «Показать настройки», отметьте «Ассистент DeepSeek», чтобы отобразить его

> **Примечание**: DLL должна находиться в подпапке `plugins`, размещение в других каталогах не позволит её загрузить.

## Описание отображаемых данных

Отображение плагина состоит из двух частей:

- **Текст левой метки**: можно изменить в TrafficMonitor: «Параметры» → «Настройки окна панели задач» → «Настройки отображаемого текста».
- **Текст правого значения**: показывает `сумма валюта`, например `100.00 CNY`. Данные баланса обновляются автоматически по расписанию.

В подменю «Отображаемые элементы» контекстного меню TrafficMonitor название плагина изменяется в зависимости от языковых настроек (например, «Ассистент DeepSeek» в русскоязычной среде).

## Открытие окна настроек плагина

Щёлкните правой кнопкой мыши по окну панели задач или главному окну TrafficMonitor, выберите «Управление плагинами» в контекстном меню, щёлкните правой кнопкой по данному плагину и выберите «Параметры» — откроется диалог настроек. В диалоге настроек можно задать API Key и другие параметры, а также просмотреть историю.

## Описание параметров

### Ключ API DeepSeek

- Введите ваш [DeepSeek API Key](https://platform.deepseek.com/api_keys) в поле ввода
- Ключ отображается в режиме маски пароля (не показывается открытым текстом)
- Под полем ввода в реальном времени проверяется формат: должен начинаться с `sk-`, за которым следуют 32 буквы и цифры
- После проверки формата нажмите кнопку «Тест API», чтобы подтвердить действительность ключа
- После успешного теста появится окно с подробной информацией о балансе, и кнопка OK станет доступной

> **Необходимо пройти тест API**, после чего нажать OK для сохранения. Если ключ пуст (отображение не требуется), можно сохранить напрямую.

### Интервал обновления (сек)

- Значение по умолчанию: **60** секунд
- Диапазон: 1 ~ 31536000
- Автоматический запрос баланса DeepSeek API через заданный интервал
- Интервал обновления должен быть больше времени ожидания запроса, иначе сохранение невозможно

### Тайм-аут запроса (сек)

- Значение по умолчанию: **10** секунд
- Диапазон: 3 ~ 60 секунд
- Максимальное время ожидания API-запроса; при превышении запрос считается неудачным
- Рекомендуется настраивать в зависимости от состояния сети: при плохом соединении можно увеличить

### Количество записей истории

- Значение по умолчанию: **1000** записей
- Диапазон: 0 ~ 10000 (0 означает, что история не ведётся)
- Новая запись добавляется только при фактическом изменении баланса (разница > 0.001), чтобы избежать избыточности
- Старые записи сверх лимита автоматически удаляются

### Автообновление

- Включено по умолчанию
- При включении список истории автоматически обновляется каждую секунду
- Щелчок в любом месте списка отключает автообновление

### Язык отображения

- Значение по умолчанию: «Авто» (автоопределение по системному языку)
- Поддерживается 12 языков: 简体中文, 繁體中文, 日本語, English, Deutsch, עברית, Magyar, Italiano, Polski, Português (Brasil), Русский, Türkçe
- После смены языка диалог применяет изменения немедленно, но для постоянного сохранения необходимо нажать «OK»

### Очистка истории

- При нажатии кнопки «Очистить историю» и подтверждении все записи истории удаляются
- Операция очистки необратима

## Сборка

### Требования к среде

- Visual Studio 2026 (набор инструментов v145)
- Windows 10 SDK или новее
- Необходимо включить опцию компиляции `/utf-8` (обязательно для исходного кода с китайскими символами)

### Быстрая сборка (рекомендуется)

```batch
build.bat
```

Автоматически собирает конфигурацию Release в соответствии с архитектурой системы и копирует полученную DLL в `C:\TrafficMonitor\plugins\`.

> **Примечание**: измените переменную `PLUGINDIR` в `build.bat` в соответствии с фактическим путём установки TrafficMonitor.

### Ручная сборка

```batch
MSBuild DeepSeekDeskBand.sln /t:Rebuild /p:Configuration=Release /p:Platform=x86
MSBuild DeepSeekDeskBand.sln /t:Rebuild /p:Configuration=Release /p:Platform=x64
```

### Выходные файлы

| Платформа | Конфигурация | Выходной файл                         |
|-----------|--------------|---------------------------------------|
| x86       | Debug        | `Debug\DeepSeekDeskBand.dll`         |
| x86       | Release      | `Release\DeepSeekDeskBand.dll`       |
| x64       | Debug        | `x64\Debug\DeepSeekDeskBand64.dll`   |
| x64       | Release      | `x64\Release\DeepSeekDeskBand64.dll` |

## Конфиденциальность и безопасность данных

Данный плагин уделяет первостепенное внимание конфиденциальности пользователей и безопасности данных:

- **Ключ API**: хранится в зашифрованном виде с помощью Windows DPAPI в локальном файле конфигурации `DeepSeekDeskBand.dat`, доступен только текущему пользователю Windows. Ключ никогда не передаётся третьим лицам.
- **Данные баланса**: получаются исключительно по HTTPS от официального API DeepSeek (`api.deepseek.com`), без промежуточных серверов.
- **История**: записи об изменениях баланса хранятся в зашифрованном виде с помощью DPAPI в локальном файле `DeepSeekDeskBand_History.dat`, доступны только текущему пользователю Windows.
- **Сетевые запросы**: отправляются только HTTPS-запросы к `api.deepseek.com` (Authorization: Bearer Token), запросы к другим доменам не выполняются.
- **Отсутствие сбора данных**: плагин не собирает и не передаёт никакие пользовательские данные или статистику использования.
- **Кэш иконки**: иконка (favicon.ico) загружается с `www.deepseek.com` только один раз при первом открытии диалога настроек и кэшируется в каталоге конфигурации плагина.

## Технические примечания

- Интерфейс плагина: TrafficMonitor API v7
- Без зависимости от MFC, реализация на чистом Win32 C++
- Управление экземпляром плагина через паттерн Singleton
- HTTPS-клиент на базе WinHTTP
- Порог обнаружения изменения баланса > 0.001 для уменьшения избыточных записей
