# עוזר DeepSeek (DeepSeekDeskBand)

![screenshot](screenshot.png)

תוסף שולחן עבודה עבור [TrafficMonitor](https://github.com/zhongyang219/TrafficMonitor). מציג מידע על יתרת חשבון DeepSeek בזמן אמת בחלון שורת המשימות, כולל היסטוריית שינויים ביתרה.

## תכונות

- שאילתה תקופתית של יתרת חשבון מ-API הרשמי של DeepSeek
- הצגת ערך היתרה והמטבע בשורת המשימות ובחלון הראשי של TrafficMonitor
- היסטוריית שינויים ביתרה (אחסון מצטבר אוטומטי, עם מגבלה מתכווננת)
- תמיכה רב-לשונית (זיהוי אוטומטי של שפת המערכת או בחירה ידנית)
- קובץ התצורה (כולל מפתח API) והיסטוריית הרשומות מוצפנים באמצעות Windows DPAPI, נגישים רק למשתמש Windows הנוכחי

## התקנה

1. הורד את קובץ ה-DLL המתאים מדף ה-[Release](https://github.com/KagurazakaYashi/TrafficMonitorPlugin-DeepSeekDeskBand/releases):
   - TrafficMonitor בגרסת x86 → `DeepSeekDeskBand.dll`
   - TrafficMonitor בגרסת x64 → `DeepSeekDeskBand64.dll`
2. העתק את קובץ ה-DLL לתיקיית `plugins` שבתיקיית התוכנה של TrafficMonitor
3. הפעל מחדש את TrafficMonitor
4. לחץ לחיצה ימנית על אזור ריק בשורת המשימות, בחר "הגדרות תצוגה" וסמן את "עוזר DeepSeek" כדי להציג

> **שים לב**: קובץ ה-DLL חייב להיות ממוקם בתוך תיקיית המשנה `plugins`. מיקום בתיקייה אחרת ימנע את טעינתו.

## תיאור תוכן התצוגה

תצוגת התוסף מורכבת משני חלקים:

- **טקסט התווית השמאלי**: ניתן לשינוי בהגדרות TrafficMonitor תחת "אפשרויות" → "הגדרות חלון שורת המשימות" → "הגדרות טקסט תצוגה".
- **טקסט הערך הימני**: מציג `ערך יתרה מטבע`, לדוגמה `100.00 CNY`. נתוני היתרה מתעדכנים אוטומטית בפרקי זמן קבועים.

בתפריט המשנה "פריטי תצוגה" של תפריט הלחיצה הימנית ב-TrafficMonitor, שם התצוגה של התוסף משתנה לפי הגדרת השפה (לדוגמה, בסביבת עברית הוא יוצג כ"עוזר DeepSeek").

## פתיחת חלון הגדרות התוסף

לחץ לחיצה ימנית על חלון שורת המשימות או החלון הראשי של TrafficMonitor, בחר "ניהול תוספים" בתפריט ההקשר, לחץ לחיצה ימנית על תוסף זה ובחר "אפשרויות" כדי לפתוח את תיבת הדו-שיח של ההגדרות. בתיבת הדו-שיח ניתן להגדיר את מפתח ה-API ואפשרויות נוספות, וכן לצפות בהיסטוריית הרשומות.

## הסבר על אפשרויות ההגדרה

### מפתח API של DeepSeek

- הזן את [מפתח ה-API של DeepSeek](https://platform.deepseek.com/api_keys) שלך בשדה הקלט
- המפתח מוצג במצב מיסוך סיסמה (לא כטקסט גלוי)
- מתחת לשדה הקלט תתבצע בדיקת תקינות בזמן אמת: חייב להתחיל ב-`sk-` ולאחריו 32 תווי אותיות וספרות
- לאחר שהפורמט תקין, לחץ על כפתור "בדיקת API" לאימות תקפות המפתח
- לאחר מעבר הבדיקה תופיע תיבת מידע עם פרטי היתרה, וכפתור האישור יהפוך לזמין

> **חובה לעבור את בדיקת ה-API** לפני שניתן ללחוץ על אישור ולשמור. אם המפתח ריק (אין צורך בתצוגה), ניתן לשמור ישירות.

### מרווח עדכון (שניות)

- ערך ברירת מחדל: **60** שניות
- טווח: 1 ~ 31536000
- היתרה מ-API של DeepSeek תיבדק אוטומטית כל מספר השניות שהוגדר
- מרווח העדכון חייב להיות גדול מזמן התפוגה של הבקשה, אחרת לא ניתן יהיה לשמור

### זמן תפוגה לבקשה (שניות)

- ערך ברירת מחדל: **10** שניות
- טווח: 3 ~ 60 שניות
- זמן ההמתנה המרבי לבקשת API. מעבר לזמן זה הבקשה תיחשב ככישלון
- מומלץ להתאים לפי מצב הרשת - ניתן להגדיל במקרה של רשת איטית

### כמות רשומות היסטוריה

- ערך ברירת מחדל: **1000** רשומות
- טווח: 0 ~ 10000 (0 משמעו ללא תיעוד היסטוריה)
- רשומה חדשה מתווספת רק כאשר חל שינוי ממשי ביתרה (הפרש > 0.001), למניעת יתירות
- רשומות ישנות החורגות מהמגבלה יימחקו אוטומטית

### רענון אוטומטי

- מסומן כברירת מחדל
- כאשר מסומן, רשימת היסטוריית הרשומות מתרעננת אוטומטית מדי שנייה
- לחיצה בכל מקום ברשימה תבטל את הרענון האוטומטי

### שפת תצוגה

- ערך ברירת מחדל: "אוטומטי" (זיהוי אוטומטי לפי שפת המערכת)
- תמיכה ב-12 שפות: 简体中文, 繁體中文, 日本語, English, Deutsch, עברית, Magyar, Italiano, Polski, Português (Brasil), Русский, Türkçe
- החלפת שפה נכנסת לתוקף מיידי בתיבת הדו-שיח, אך נשמרת לצמיתות רק בלחיצה על כפתור "אישור"

### ניקוי היסטוריה

- לחץ על כפתור "נקה היסטוריה" ואשר כדי למחוק את כל רשומות ההיסטוריה
- פעולת הניקוי אינה הפיכה

## הידור (קומפילציה)

### דרישות סביבה

- Visual Studio 2026 (ערכת כלים v145)
- Windows 10 SDK ומעלה
- חובה להפעיל את אפשרות ההידור `/utf-8` (נדרש עבור קוד מקור בסינית)

### הידור מהיר (מומלץ)

```batch
build.bat
```

מהדר אוטומטית את תצורת ה-Release בהתאם לארכיטקטורת המערכת הנוכחית, ומעתיק את קובץ ה-DLL שנוצר אל `C:\TrafficMonitor\plugins\`.

> **שים לב**: יש לשנות את המשתנה `PLUGINDIR` בקובץ `build.bat` בהתאם לנתיב ההתקנה בפועל של TrafficMonitor.

### הידור ידני

```batch
MSBuild DeepSeekDeskBand.sln /t:Rebuild /p:Configuration=Release /p:Platform=x86
MSBuild DeepSeekDeskBand.sln /t:Rebuild /p:Configuration=Release /p:Platform=x64
```

### קבצי פלט

| פלטפורמה | תצורה  | קובץ פלט                              |
|----------|--------|---------------------------------------|
| x86      | Debug  | `Debug\DeepSeekDeskBand.dll`         |
| x86      | Release| `Release\DeepSeekDeskBand.dll`       |
| x64      | Debug  | `x64\Debug\DeepSeekDeskBand64.dll`   |
| x64      | Release| `x64\Release\DeepSeekDeskBand64.dll` |

## פרטיות ואבטחת מידע

תוסף זה מייחס חשיבות רבה לפרטיות המשתמש ואבטחת המידע:

- **מפתח API**: מוצפן באמצעות Windows DPAPI ומאוחסן בקובץ התצורה המקומי `DeepSeekDeskBand.dat`, נגיש רק למשתמש Windows הנוכחי. המפתח לעולם אינו מועלה לשום צד שלישי.
- **נתוני יתרה**: מתקבלים אך ורק דרך HTTPS מ-API הרשמי של DeepSeek (`api.deepseek.com`), ללא מעבר דרך שרתי ביניים כלשהם.
- **היסטוריית רשומות**: רשומות שינויי היתרה מוצפנות באמצעות DPAPI ומאוחסנות בקובץ המקומי `DeepSeekDeskBand_History.dat`, נגישות רק למשתמש Windows הנוכחי.
- **בקשות רשת**: נשלחות בקשות HTTPS רק אל `api.deepseek.com` (Authorization: Bearer Token), ללא פניה לשמות מתחם אחרים.
- **ללא איסוף נתונים**: תוסף זה אינו אוסף או מעלה כל נתוני משתמש או סטטיסטיקות שימוש.
- **מטמון סמלים**: סמל האתר (favicon.ico) מוריד פעם אחת מ-`www.deepseek.com` רק בפתיחה הראשונה של תיבת דו-שיח ההגדרות, ונשמר במטמון בתיקיית תצורת התוסף.

## פרטים טכניים

- ממשק תוסף: TrafficMonitor API v7
- ללא תלות ב-MFC, מימוש טהור ב-Win32 C++

- לקוח WinHTTP HTTPS
- סף זיהוי שינוי ביתרה > 0.001, להפחתת רשומות מיותרות
